Gate Thresholds

    
Move / Trigger thresholds use parameters 0x10..0x1F. Still / Hold thresholds use parameters 0x20..0x2F.

    
This table uses the official HLK calibrator scale from 0 to 90.

    
Conversion used behind the scenes: HLK = 10 * log10(raw), raw = 10^(HLK / 10). Example: raw 1000000 displays as HLK 60.

    
Gate coverage is approximately 0.7 m per gate: Gate 0 is 0.0-0.7 m, Gate 1 is 0.7-1.4 m, Gate 2 is 1.4-2.1 m, and so on.

    
Gate actions: Disable sets a gate to 90/90, which is the current convention for turning that gate off. Enable restores a usable value if a gate was disabled, and Reset returns that gate to a neutral 60/60 starting point.
